jQuery - Example (Dave)

jQuery

$(document).ready(function() {
    $('button').click(function() {
        var name = $('[name="your-name"]').val();
        $('#mydiv').html('Hi ' + name);
    });
});

Markup

<input type="text" name="your-name" />
<button>Say Hi</button>
<div id="mydiv"></div>
